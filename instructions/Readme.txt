Instruction to run the files:

1. All the python packages mentioned in "requirements.txt" are installed (preferably in a virtual environment) to run the program. 
2. The "textAnalysis.py" file contains the main runner methods. In the terminal the preogram can be run by executing the "python3 textAnalysis.py" command.
3. The program requires as input two author names separated by a comma "," : (Firstname Lastname, Firstname Lastname )
4. If the API cannot find workd on any of the entered author names, new names must be provided with the same format as above.
